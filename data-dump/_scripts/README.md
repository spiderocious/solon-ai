# Helper Scripts

Tiny standalone data-collection helpers. All scripts:
- Are self-contained (Python 3 or Node.js).
- Use only standard library + `requests` (Python) or built-in `fetch` (Node).
- Do nothing outside of reading from the web and writing into `data-dump/`.
- Are one-shot; they do not run as services or daemons.
